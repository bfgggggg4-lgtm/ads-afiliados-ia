COMO USAR:

1. Extraia este ZIP.
2. No Netlify, abra seu projeto.
3. Faça novo deploy manual enviando a pasta ads-afiliados-ia.
4. Vá em Project configuration.
5. Vá em Environment variables.
6. Adicione:
   Nome: OPENAI_API_KEY
   Valor: sua chave OpenAI
7. Salve.
8. Faça Redeploy.
9. Abra o site e teste.

Nunca coloque sua chave OpenAI no script.js.
